# Defect Report

<details>
<summary><strong>🐞 DEF-001: Direct Navigation / Refresh on Category Routes Returns 404</strong></summary>

### Summary
Accessing category routes directly through the browser URL or refreshing the page results in a 404 error.

### Steps to Reproduce
1. Open application
2. Navigate to Popular category
3. Refresh browser

### Expected Result
Category page should load successfully.

### Actual Result
404 Page Not Found is displayed.

### Severity
High

### Priority
High

### Status
Open

</details>

<details>

### Actual Result

Medium

Medium
